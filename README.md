# Simple Interest Calculator

This project contains a Bash script that calculates simple interest.

## Description
The simple interest is calculated using the formula:

Simple Interest = (Principal × Rate × Time) / 100

The script takes the principal amount, rate of interest, and time period as input from the user and calculates the simple interest.

## Technologies Used
- Bash Shell Script
- Git & GitHub

## Purpose
This project demonstrates basic GitHub repository structure and simple shell scripting for financial calculations.
